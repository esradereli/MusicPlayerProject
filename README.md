# MusicPlayerG7

MusicPlayerG7 is a desktop-based music management application developed using Java Swing and MySQL.

The project allows users to manage songs and artists through a graphical user interface while practicing database integration, CRUD operations, and object-oriented programming concepts.

![Logo](logo.jpeg)

---

## Features

- User Login & Registration
- Session Management
- Password Change System
- Add / Update / Delete Songs
- Add / Update / Delete Artists
- Dynamic Search System
- MySQL Database Integration
- JTable Based Data Listing
- Delete Confirmation Mechanism
- Error Handling & Input Validation

---

## Technologies Used

- Java
- Java Swing
- JDBC
- MySQL
- FlatLaf
- NetBeans IDE

---

## Database Tables

The application uses three main tables:

- Users
- Songs
- Artists

The system performs full CRUD operations using JDBC and MySQL integration.

---

## Main Modules

- Authentication System
- Song Management
- Artist Management
- Session Management
- Database Connection Layer

---

## How to Run

1. Open the project in NetBeans IDE
2. Configure the MySQL database connection
3. Add required libraries
4. Build and run the project

Or run directly with:

```bash
java -jar MusicApp.jar
```

## Purpose of the Project

The purpose of this project is to improve practical knowledge in:

- Object-Oriented Programming
- GUI Development

through a functional music management system.
